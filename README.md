# 🍽️ Zomato Data Analysis

## 📊 Overview :

This project analyzes Zomato restaurant data to understand customer preferences, restaurant performance, and key business trends. The goal is to generate insights that can help improve decision-making in the food and hospitality industry.

## 📁 Dataset :

The dataset contains restaurant-related information such as:

  * Location and city
  * Ratings and reviews
  * Cost for two
  * Cuisines offered

## 🛠️ Tools & Technologies :

  * Python (Pandas, NumPy)
  * Matplotlib / Seaborn

## ⚙️ Project Steps :

  * Data Loading – Imported dataset using Pandas
  * Exploratory Data Analysis (EDA) – Analyzed distributions and trends
  * Data Cleaning – Handled missing values, removed duplicates, and corrected data types
  * Data Visualization – Created charts to identify patterns and insights

## 📈 Results :

  * Identified top-rated restaurants and popular cuisines
  * Analyzed cost vs rating relationships
  * Discovered location-based trends in restaurant performance

## ✅ Conclusion :

This project demonstrates how data analysis can be used to understand customer preferences and improve business strategies in the restaurant industry.
